## 🔧 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/SolomonLincolnJr/webdevelopment.git
cd webdevelopment

# Install dependencies
npm install

# Start the server
npm start
```

### Development Commands

```bash
# Start development server
npm run dev

# Run tests
npm test

# Build for production
npm run build

# Deploy to Railway
npm run deploy
```
